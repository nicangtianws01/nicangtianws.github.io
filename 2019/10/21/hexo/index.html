hexo+github博客教程

hexo+github博客教程

十月 21, 2019
  • github
  • hexo

安装了nodeJS、安装并配置好git：
一、设置全局账号：
命令：git config –global user.name “用户名”
git config –global user.email “邮箱”

二、添加本机sshkey
SSh key

1.生成ssh key：
命令：ssh-keygen -t rsa -C "邮箱"

生成的ssh key在${user.home}/.ssh文件夹中
github.com帐号settings中添加ssh key，将生成的id_rsa.pub文件内容复制过去

2.查看是否添加成功：
命令：ssh -T git@github.com

3.github创建名为：用户名.github.io的仓库

安装hexo：
打开git bash并执行：npm install -g hexo

初始化：
选定hexo博客文件目录存放位置，右键打开git bash，执行：hexo init

更换主题：
在themes目录打开git bash执行：git clone https://github.com/Fechin/hexo-theme-diaspora

更改hexo的配置文件：_config.yml
deploy:
type: git
repo: git@github.com:用户名/用户名.github.io.git
branch: master

上传到github：
hexo g生成博客
hexo d上传到github

修改文章模板（scaffolds文件夹下）：
—
title: My awesome title
date: 2016-10-12 18:38:45
categories:
- 分类1
- 分类2
tags:
- 标签1
- 标签2
mp3: http://domain.com/awesome.mp3
cover: http://domain.com/awesome.jpg
—

发布文章：
hexo new “文章标题（之后可以在文件中修改）”

创建分类页:
hexo new page categories

创建标签页:
hexo new page tags
